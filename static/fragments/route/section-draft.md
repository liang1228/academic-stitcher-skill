# Route: Section Draft

Use when drafting or rebuilding a manuscript section from claims, notes, figures, tables, or Chinese drafts.

## Required Moves

1. Build a one-sentence argument before prose.
2. Select the section fragment and paper-type fragment.
3. Assign one job to each paragraph.
4. Draft from evidence outward.
5. Return a claim-evidence-boundary map.
6. Run a source-fidelity pass: remove any detail that is clearer, narrower, or more specific than the supplied notes.

## Boundary

Do not fill missing results, citations, mechanisms, examples, or applications with invented content. Use placeholders such as `[Evidence needed: ablation for module X]`. If the source only says an application is possible, keep it at that level or mark the specific application as a proposal.
