# Paper Type: Proposal Or Thesis

Use for opening reports, graduate theses, advisor-facing plans, graduation-focused small papers, and defense preparation.

## Reader Question

Is the work feasible, attributable, useful for the user's target, and defensible under time constraints?

## Argument Chain

Graduation or advisor target -> current resources -> feasible topic -> inherited baseline -> work packages -> evidence plan -> schedule -> risk controls.

## Required Checks

- Minimum viable research product.
- Advisor decision needed.
- Dataset/code/compute access.
- Authorship and lab-lineage boundary.
- Timeline with buffer for failed runs.
