# Paper Type: Methods

Use for protocols, measurements, datasets, benchmarks, tools, or reusable technical procedures.

## Reader Question

Does the method work, is it better than alternatives, and can others reproduce it?

## Argument Chain

Existing workflow limitation -> proposed method -> validation -> comparison -> reproducibility -> use cases.

## Required Evidence

- Clear input/output and procedure.
- Validation on representative settings.
- Fair comparison with alternatives.
- Reproducibility details and limitations.
