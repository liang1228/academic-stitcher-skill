# Paper Type: Review

Use for literature reviews, survey sections, related-work-heavy plans, or state-of-field synthesis.

## Reader Question

What is known, where do methods disagree, and what remains open?

## Argument Chain

Scope -> taxonomy -> method families -> evidence patterns -> disagreement -> open problem -> future direction.

## Rules

- Organize by argument, not by paper order.
- State inclusion/exclusion logic when the review is formal.
- Separate accepted evidence from speculation.
- Do not use a review to imply unverified novelty for a later method.
