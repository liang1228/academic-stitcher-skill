# Behavioral Boundaries & Anti-Patterns

## Behavioral Boundaries

| Boundary | Correct Action |
| :--- | :--- |
| Do not invent non-existent technologies | Rely only on scientifically validated mechanics and principles. |
| Do not use generic descriptions | Specify the exact physical, chemical, or algorithmic mechanism of action. |
| Do not include conversational filler | Omit introductory or concluding meta-commentary (e.g., "Here is your plan"). |
| Do not propose surface-level stitching | Provide a clear, detailed technical bridge between Target and Catalyst. |

## Anti-Pattern Fixes

| Symptom | Anti-Pattern | Corrected Formulation |
| :--- | :--- | :--- |
| **Hype & Buzzwords** | "This revolutionary approach completely paradigm-shifts X..." | "Integrating [Catalyst] into [Scenario] reduces error rate by 40%." |
| **The "Recent Years" Trap** | "In recent years, the development of GNNs has attracted attention..." | "Graph Neural Networks capture non-local atomic interactions in crystal lattices." |
| **Incremental Stacking** | "Applying standard deep learning to optimize materials..." | "Deploying Multi-Scale PINNs to map electrolyte interface phase-change dynamics." |
