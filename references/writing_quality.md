# Writing Quality Guidelines

## Principle: Fact-Driven Precision

The goal is to produce academic prose that is indistinguishable from top-tier peer-reviewed literature. Avoid the "AI signature" by following these rules.

## Banned AI Phrases & Structures

| Avoid | Replace With |
| :--- | :--- |
| "In recent years..." | [Direct statement of fact or trend] |
| "It is worth noting that..." | [State the fact directly] |
| "A testament to..." | [Evidence indicates...] |
| "In conclusion/Summary..." | [End with a forward-looking implication] |
| "Crucial/Vital/Essential" | [Specific technical reason why it matters] |
| "Harnessing the power of..." | "Utilizing [X] to [Y]..." |

## Sentence Structure Rules

1.  **No Fluff**: Every sentence must convey new technical information. If a sentence can be removed without losing the core argument, remove it.
2.  **No Hype**: Remove subjective adjectives like "groundbreaking", "revolutionary", or "unprecedented". Let the data and methodology speak for themselves.
3.  **Active Voice in Methods**: Use active voice when describing the proposed stitching mechanism (e.g., "We integrate [Catalyst] into [Scenario] to solve [Object]...").
4.  **Avoid Nominalization**: Instead of "The implementation of the system was performed", use "We implemented the system".
