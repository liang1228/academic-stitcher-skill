# SKILL: Document Summarization

## Steps
1. Receive the document text as input.
2. Analyze the text to identify the main topics, key arguments, and supporting details.
3. Generate a concise summary that captures the essential information of the original document.
4. Output the summary.

## Constraints
- The input document should be in a readable text format.
- The summary should be significantly shorter than the original document while retaining core meaning.
- Maintain factual accuracy and avoid introducing external information.

## Anti-patterns
- Generating summaries that are longer than the original document.
- Including personal opinions or interpretations not present in the source text.
- Failing to capture the main points of the document.
- Summarizing only a portion of the document without indicating it.
